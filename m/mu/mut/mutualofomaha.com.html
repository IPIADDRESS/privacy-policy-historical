Mutual places Internet cookies on our Users' hard drives. A cookie is a small text file that is sent from Mutual's Web server to a Users' personal computer. This file contains information about a User's visit to Mutual's Web site that only Mutual's server can read. The cookies are stored on the User's hard drive and are not stored, in any form, by Mutual. Cookies are used throughout the log-in session to enhance a User's return visit to Mutual's Web site. They are also used, when applicable, to verify the User's authorization to access certain Mutual Web pages (e.g., a broker's access to the "Mutual e-net") so that the Users are not required to enter a user name and password every time they access a different page. Cookies may also provide a record of a User's visit, including information such as their User ID, password, time of last visit, and Web pages viewed.
