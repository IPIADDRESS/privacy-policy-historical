Data Privacy Policy

Scope

This data privacy statement is intended to inform the user as to the nature, extent and purpose of collecting and using personal data through this website by the vendor responsible for the site (fabulabs GmbH, Pappelallee 78/79, 10437 Berlin Germany).

Access Data / Server Log Files

The vendor (or webspace provider) collects data on access to the site (so-called server log files). Access data includes:

Name of the website accessed; files accessed and the date and time they were accessed; volume of data transmitted; notification of successful retrieval; browser type and version; user's operating system; referring URL (page previously visited), IP address and the requesting provider.

The vendor uses the log files solely for statistical analysis for the purpose of operating, securing and improvement of the site. However, the vendor reserves the right to subsequently review the log files when there is specific evidence giving rise to the reasonable suspicion of illegal use.

Cookies

Cookies are small files that allow specific information to be stored relating to the user's specific access device (PC, smartphone or the like). They serve, on the one hand, to benefit the user by making the site more user friendly (e.g. by storing login data). On the other hand, they also collect statistical data on website use in order to analyze that use for the purpose of improving the site. The user can influence the way cookies are stored and used. Most browsers have an option that allows the user to restrict or block the use of cookies. However, it must be noted that doing so will restrict use and in particular user friendliness of the site.

You can manage many online ad cookies from private companies via the US website, http://www.aboutads.info/choices/ or the EU site, http://www.youronlinechoices.com/uk/your-ad-choices/.

Use of Personal Data

Personal data is information that can be used to identify a particular individual person, in other words, information that can be traced back to a specific individual person. This includes name, email address or telephone number. But personal data can also include information about preferences, hobbies, memberships or websites visited. Personal data may be collected and used by the vendor only where allowed by law or where the user has provided consent to the collection of his/her data.

Registration

Data provided during registration will be employed in association with use of the site, in relation to site technical information and for sending newsletters and web mailings, where consent to such has been provided. Users can obtain information by email about the site or about registration, such as changes to the site or technical issues. The data being collected is entered in the fields that are filled in during registration. These include, for example, first name, last name, sex, email address, IP address and the date and time of registration. If a user registers through the authentication feature on Facebook, the user's Facebook ID will be stored, as well as, where applicable, information about the user's sex, date of birth, likes and friends list. The scope of data collected may be obtained through the Facebook login dialogue. The specific data that Facebook makes available can be found at the following link: https://www.facebook.com/about/privacy/your-info-on-other.

Facebook makes the determination based on the provisions of its own data privacy policy as to which data will always be publically available and which data you can make private via your privacy settings. For the details, go to: https://www.facebook.com/about/privacy/your-info.

If a user no longer wishes to make use of the Facebook application, the user can terminate the user agreement. To do so, the user must go to the so-called "App Center" in his/her Facebook settings and terminate the link between the app and his/her Facebook profile. Directions on how to do this can be found here: https://www.facebook.com/help/121070141307903. Please note: Deleting the app does not delete the user's profile data from the vendor's server. To delete this data, the user must submit a separate withdrawal of consent to the use of his/her data to the email address shown in the section on "Withdrawals".

Consent to Receive Ads

The user agrees to receive from the vendor messages containing informational or promotional materials from various business sectors via electronic channels (e.g. email, notifications or mobile push-notifications). However, this applies only where the user has explicitly provided consent (e.g. by a consent box). The user can withdraw such consent at any time (contact email address: see "Withdrawals").

Withdrawals, Modifications, Notifications and Updates

The user has the right to receive upon request and at no cost information about personal data being kept on him/her. In addition, the user has the right to be informed of incorrect data or if his/her personal data has been blocked or deleted, where there is no legal obligation to retain such data. Requests should be sent by email to the following email address: info@mein-wahres-ich.de.
