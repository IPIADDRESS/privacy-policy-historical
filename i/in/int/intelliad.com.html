IP Address

The IP address is transmitted at every server request, so that the server knows where the corresponding answer needs to be sent. The IP address is provided by an Internet Service Provider (ISP) as soon as a user connects with the internet and the ISP can understand which IP address is assigned to which client at a certain time. As long as the IP address is stored, the identity of the user can theoretically be determined via the ISP. Therefore, we save the IP address in (anonymized) form and only use them to recognize the session, for geo-localization (until city –level) and for the defense of attacks. The IP address is deleted instantly so that the collected data are anonymous and the identification of the user’s identity via ISPs is no longer possible.

Cookies and Clear GIFs

What are Cookies?

Cookies are data files that are placed on the internet user’s PC by webservers and save basic information (for example user interests). The use of cookies on this site offers users a user-friendly visit based on the measurement of the user behavior on the site. There are two types of cookies: session-cookies and persistent cookies. Session-cookies are only available during the online session of an internet user. They disappear from the user’s PC as soon as the user closes the browser software. Persistent cookies stay on the user’s PC even if the browser is closed. The cookies used on the intelliAd site are not individual-related. They only recognize the user’s browser. The following cookies are used by intelliAd: - Last Click (for session timeout, lifetime: one session) – Session Cookie (for session recognition, lifetime: one session) – Ever-Cookie (for recognition of new/steady customers)
