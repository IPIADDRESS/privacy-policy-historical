CODE Privacy Policy

CODE Magazine takes your privacy seriously. Please read the following to learn about our privacy policy.

What This Privacy Policy Covers

  • This policy covers how CODE Magazine treats personal information that CODE Magazine collects and receives, including information related to your past use of CODE Magazine products and services. Personal information is information about you that is personally identifiable like your name, address, email address, or phone number, and that is not otherwise publicly available.
  • This policy does not apply to the practices of companies that CODE Magazine does not own or control, or to people that CODE Magazine does not employ or manage.

Information Collection and Use

  • CODE Magazine collects personal information when you register with CODE Magazine and when you visit CODE Magazine pages. CODE Magazine may combine information about you that we have with information we obtain from business partners or other companies.
  • When you register we ask for information such as your name, email address, zip code, and other personal information. Once you register with CODE Magazine and sign in to our services, you are not anonymous to us.
  • CODE Magazine collects information about your transactions with us and with some of our business partners, including information about your use of services we offer.
  • CODE Magazine automatically receives and records information on our server logs from your browser, including your IP address, CODE Magazine cookie information, and the page you request.
  • CODE Magazine uses information for the following general purposes: to customize the advertising and content you see, customize the content you see, fulfill your requests for products and services, improve our services, contact you, conduct research, and provide anonymous reporting for internal and external clients.

Information Sharing and Disclosure

  • CODE Magazine does not rent, sell, or share email addresses unless the owner of the email address indicated interest in partner announcements.
    • Users who have previously opted in to receive such information can opt-out at any point.
  • CODE Magazine collects demographics information about subscribers and other visitors to the CODE Magazine web site on an ongoing basis. Demographic profiles may be shared with our partners and advertisers on an aggregate bases.
  • We reserve the right to use our mailing lists for marketing purposes. On occasion, this may include marketing campaigns that include partner organizations.
  • We may provide the information to trusted partners who work on behalf of or with CODE Magazine under confidentiality agreements. These companies may use your personal information to help CODE Magazine communicate with you about offers from CODE Magazine and our marketing partners. However, these companies do not have any independent right to share this information.
  • CODE Magazine may give out personal information as a response to subpoenas, court orders, or legal process, or to establish or exercise our legal rights or defend against legal claims;
  • CODE Magazine may share personal information when we believe it is necessary to share information in order to investigate, prevent, or take action regarding illegal activities, suspected fraud, situations involving potential threats to the physical safety of any person, violations of CODE Magazine's terms of use, or as otherwise required by law.
  • CODE Magazine reserves the right to transfer information about you if CODE Magazine is acquired by or merged with another company. In this event, CODE Magazine will notify you before information about you is transferred and becomes subject to a different privacy policy.
  • CODE Magazine displays targeted advertisements based on personal information. Advertisers (including ad serving companies) may assume that people who interact with, view, or click on targeted ads meet the targeting criteria.
    • CODE Magazine does not provide any personal information to the advertiser when you interact with or view a targeted ad. However, by interacting with or viewing an ad you are consenting to the possibility that the advertiser will make the assumption that you meet the targeting criteria used to display the ad.  

Cookies

  • CODE Magazine may set and access CODE Magazine cookies on your computer.
  • CODE Magazine reserves the right to use web beacons to access CODE Magazine cookies inside and outside our network of web sites and in connection with CODE Magazine products and services.

Your Ability to Edit Your Account Information and Preferences

  • You can edit your CODE Magazine Account Information at any time. This includes email preferences.
  • CODE Magazine may change the available options for email preferences at any time. This includes consolidation of email categories and newsletters. CODE Magazine will not add people to mailing lists. All CODE Magazine mailing lists are strictly opt-in lists.
  • We reserve the right to send you certain communications relating to the CODE Magazine service, such as service announcements, administrative messages and the CODE Magazine invoices, that are considered part of your CODE Magazine account, without offering you the opportunity to opt-out of receiving them.

Confidentiality and Security

  • We limit access to personal information about you to employees who we believe reasonably need to come into contact with that information to provide products or services to you or in order to do their jobs.
  • We have physical, electronic, and procedural safeguards to protect personal information about you.
  • CODE Magazine takes great care to adhere to security best practices. As an example, CODE Magazine avoids storing confidential information (such as passwords) whenever possible. However, CODE Magazine does not guarantee that such information will never be stored in any form. 

Changes to this Privacy Policy

  • CODE Magazine may update this policy. We will notify you about significant changes in the way we treat personal information by sending a notice to the primary email address specified in your CODE Magazine account or by placing a prominent notice on our site.

If you have further questions about this privacy policy, please send an email to info@code-magazine.com.
