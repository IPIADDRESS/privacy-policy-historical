Privacy Policy


 Personal information collected on CPU-World site: 
  • For all users registered in the forum we collect e-mail address and other information provided by users during registration.
  • We use cookies to store username and logged-in status of all users registered in the forum.
  • We log search text submitted into site search, S-spec search and CPU search tools.
  • We log text that is submitted into CPU identification tool.
  • For each received page request our server automatically logs date and time of the request, visitor's IP address, accessed URL, and browser type.
  • Many CPU-World pages include advertisements supplied by external sites, such as Google or Amazon. These sites track all clicks on the ads, they may also track visited pages for each site visitor. Please consult privacy policy for those sites for more information.

 How personal information is used 
  • Personal information and cookies are used to allow registered users to access the forum. Some information about registered users, such as first and last name, is displayed in the forum members list.
  • We can use registered user's e-mail address to contact him/her in cases if there is a problem with user's account.
  • We do not share personal information of our users with third parties.
  • Information about all visited pages is used to track abusive visitors, usually automated software such as spam bots, crawlers, etc. This information is also used to generate site usage reports.
  • Logged search text and identification text are used to improve the quality of search and identification tools. This information may be also used to track abusive visitors, such as visitors trying to post spam or break into the site.

If you have any question about privacy policy please contact us.
