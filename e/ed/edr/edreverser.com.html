1.3 Cookies, Web Beacons, and Other Info Collected Using Technology.

The ED Reverser System currently uses cookie and web beacon technology to associate certain Internet-related information about an individual with information about the individual in our database. Additionally, The ED Reverser System may use other new and evolving sources of information in the future (together, "Technology Information").

(a) Cookies.

A cookie is a small amount of data stored on the hard drive of the individual's computer that allows The ED Reverser System to identify the individual with his or her corresponding data that resides in The ED Reverser System's database. You may read more about cookies at http://cookiecentral.com. Individuals who use the Websites need to accept cookies in order to use all of the features and functionality of the Websites.

(b) Web Beacons.

A web beacon is programming code that can be used to display an image on a web page (by using an programming function -- see http://truste.org for more information), but can also be used to transfer an individual's unique user identification (often in the form of a cookie) to a database and associate the individual with previously acquired information about an individual in a database. This allows The ED Reverser System to track certain web sites an individual visits online. Web beacons are used to determine products or services an individual may be interested in, and to track online behavioral habits for marketing purposes. For example, The ED Reverser System might place, with the consent of a third party website, a web beacon on the third party's website where fishing products are sold. When Bill, an individual listed in The ED Reverser System's database, visits the fishing website, The ED Reverser System receives notice by means of the web beacon that Bill visited the fishing site, and The ED Reverser System would then update Bill's profile with the information that Bill is interested in fishing. The ED Reverser System may thereafter present offers of fishing related products and services to Bill. In addition to using web beacons on web pages, The ED Reverser System also uses web beacons in email messages sent to individuals listed in The ED Reverser System's database.

(c) New Technology.

The use of technology on the Internet, including cookies and web beacons, is rapidly evolving, as is The ED Reverser System's use of new and evolving technology. As a result, The ED Reverser System strongly encourages individuals to revisit this policy for any updates regarding its use of technology.
