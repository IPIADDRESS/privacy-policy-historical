5    Penggunaan cookies

5.1    Apabila anda berinteraksi dengan kami di laman web kami, kami mungkin mengumpul atau menganalisis maklumat yang mana individu-individu tidak dapat dikenal pasti ("Maklumat Agregat"), seperti bilangan pengguna dan kekerapan penggunaan, bilangan paparan halaman (atau pertunjukan halaman) yang berlaku di laman web dan  tempat masuk dan keluar biasa ke dalam laman web .

5.2    Kami menggunakan "cookies", di mana fail data kecil dikirimkan ke penyemak imbas anda untuk menyimpan dan mengesan Maklumat Agregat tentang anda apabila anda memasuki laman web kami. “cookie” ini digunakan untuk mengesan maklumat seperti bilangan pengguna dan kekerapan penggunaan, profil pengguna dan laman kegemaran mereka.

5.3      Kami mungkin menggunakan sebuah syarikat bebas ("Syarikat Penyelidikan Pasaran Pihak Ketiga") untuk mengukur dan menganalisis penggunaan Internet di seluruh laman web kami. Kami menggunakan perkhidmatan Syarikat Penyelidikan Pasaran Pihak Ketiga untuk mengumpul maklumat teras berikut mengenai penggunaan laman web kami, termasuk:

(a)    Bilangan paparan halaman (atau pertunjukan halaman) yang berlaku di laman web kami;

 (b)    Bilangan pelawat unik ke laman web kami;

 (c)    Berapa lama pelawat unik ini (secara purata) menghabiskan masa di laman web kami apabila mereka melawat; dan

 (d)    Tempat masuk dan keluar biasa ke dalam laman web kami.

5.4      Maklumat ini dikumpulkan oleh Syarikat Penyelidikan Pasaran Pihak Ketiga dan diberikan kepada kami untuk membantu dalam menganalisis penggunaan laman web kami. Data tersebut juga boleh diakses oleh pertubuhan-pertubuhan media dan syarikat penyelidikan, untuk tujuan menjalankan perbandingan industri dengan portal Internet yang lain.

5.5    Halaman di laman web kami boleh dikodkan dengan perisian yang membolehkan Syarikat Penyelidikan Pasaran Pihak Ketiga untuk mengesan pelawat ke laman web kami.

5.6    Sekiranya anda ingin mematikan cookies yang berkaitan dengan teknologi-teknologi ini, anda boleh berbuat demikian dengan menukar tetapan pada pelayar anda. Walau bagaimanapun, anda mungkin tidak dapat memasuki bahagian tertentu di laman web kami.

6    Laman Pihak Ketiga

6.1    Laman web kami mungkin mengandungi pautan ke laman web lain yang dikendalikan oleh pihak ketiga, seperti rakan-rakan perniagaan kami. Kami tidak bertanggungjawab untuk amalan privasi laman web yang dikendalikan oleh pihak ketiga yang dihubungkan ke laman web kami. Kami menggalakkan anda untuk belajar tentang dasar privasi laman web pihak ketiga tersebut. Sesetengah daripada laman web pihak ketiga mungkin dijenamakan bersama dengan logo atau cap dagangan kami, walaupun mereka tidak dikendalikan atau disenggarakan oleh kami. Sebaik sahaja anda meninggalkan laman web kami, anda patut menyemak terma-terma terpakai, syarat-syarat dan dasar-dasar laman web pihak ketiga untuk menentukan bagaimana mereka akan menangani apa-apa maklumat yang mereka kumpul daripada anda.
