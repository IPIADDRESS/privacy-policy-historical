5. Cookies

A cookie is a small data file that consists of information sent by a web server to a web browser such as Internet Explorer or Firefox and which is stored by that browser on the computers hard drive. The information stored is sent back to the web server each time the browser requests a tutorial page from the server.

This action enables the web server to identify and track the web browsers activities. We use session cookies on the website to keep track of you whilst you navigate through our website. Session cookies are temporary files that are automatically deleted from your computers hard drive when you close the browser.

This website: Electronics-tutorials.ws, uses Google Analytics as a statistical tool to gather information in order for us to analyse the use of the website to improve the user experience. Google Analytics stores this information as a cookie (a little data file) on the visitors computer.

You can find more information about how Google uses cookies from the following link: How Google uses Cookies. You can also find information about the new European Directive and the use of Cookies at: http://ec.europa.eu/ipg/basics/legal/cookies/index_en.htm who have implemented this new “Cookie Law” legislation.

The statistical information generated by Google Analytics relating to our website is used to create internal reports to help improve the content and user experience of our electronics tutorials website. Google’s privacy policy regarding Google Analytics is freely available at: http://www.google.com/privacy/privacy-policy

You can allow, warn, block or remove cookies installed on your computer any time by using the setting for the browser installed on your computer. However, blocking all cookies may have a negative impact upon the usability of the website any you may not be able to use certain site features or functions.
