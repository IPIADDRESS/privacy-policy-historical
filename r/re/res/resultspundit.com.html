www.resultspundit.com makes use of the data contained in log files like most of the other websites. This data includes internet protocol ( IP ) addresses, type of website browsers( such as Internet explorer or Firefox or chrome), Internet Service Provider( ISP like Airtel or Tata Indicom ), date/time stamp of user’s visites, entry/exit pages, number of pageviews per use & and total number of users to gather statistics about the website’s users in order to administer the site effectively.

www.resultspundit.com does use cookies to store information about user’s preferences, record user-specific information on which pages the user access or visit, customize Web page content based on visitors browser type or other information that the visitor sends via their browser.
