What is NinjaManager?

NinjaManager is a free browser based MMORPG set in the world of the Naruto manga and anime series. In short, a free naruto online game.

A cookie is a piece of information in the form of a very small text file that is placed on an internet user's hard drive. It is generated by a web page server, which is basically the computer that operates a web site. The information the cookie contains is set by the server and it can be used by that server whenever the user visits the site. We use cookies to automatically log you in (upon choosing that option), store preferences, record past activity at a site in order to provide better service when visitors return to our site, customize Web page content based on visitors' browser type or other information that the visitor sends.
