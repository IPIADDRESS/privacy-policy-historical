Registration is not required to use this site, with the exception of the Video Center section where a user must first complete the registration form. We do not use this information, but it is passed on to RealEnglisn.com to contact the user about services on their site for which the user has expressed interest. 1-language.com also has Discussion Forums which have optional registration. For more details see "Profile" below.


Information Use
All information collected by 1-language.com is of a non-personal nature except the Discussion Forums and the Video Center. 1-language.com records aggregate information such as number of users, location, server type, and method of arriving at 1-language.com. The Discussion Forums allow users to optionally register information such as username, password, location, and interests. The Discussion Forums are run using PHPBB2 software which uses a MySQL database, and so present a very small security risk from hacking attempts. The Video Center requires a valid email address, username and password which are not used by 1-language.com, but are passed on to RealEnglish.com.

Profile
1-language.com collects profiles from our discussion forums which use PHPBB2 software. Registration for these forums is not required, and if a users chooses to register only an email address is required. Cookies
A cookie is a piece of data stored on the user¡¯s computer tied to information about the user. Usage of a cookie is in no way linked to any personally identifiable information while on our site. We use both session ID cookies and persistent cookies. For the session ID cookie, once users close the browser, the cookie simply terminates. A persistent cookie is a small text file stored on the user¡¯s hard drive for an extended period of time. Persistent cookies can be removed by following Internet browser help file directions.

Some of our business partners use cookies on our site (for example, advertisers). However, we have no access to or control over these cookies, once we have given permission for them to set cookies for advertising.

This privacy statement covers the use of cookies by 1-language.com only and does not cover the use of cookies by any advertisers.
