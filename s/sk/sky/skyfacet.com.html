  Cookies:
   
  Sky Facet uses cookies to give our customers a better shopping experience. A cookie is a signal given to a web browser (i.e. internet explorer) by a web server. The browser stores the signal in a text format. The signal is then sent back to the server each time the browser requests a page from the server.
