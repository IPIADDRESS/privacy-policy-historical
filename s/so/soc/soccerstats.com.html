       - Use of cookies
    A "cookie" is a small piece of information, a snippet of text, that is sent from a website's server and stored on a web browser so it can later be read back from that browser. Typical cookie usage include enhancing the user's browsing experience. For example, the use of a cookie can allow a web server to display a web page customised with prior settings and preferences set by the user during a previous visit to the website. Other possible uses include some e-commerce sites, which may use cookies to "remember" the products placed by the user in their shopping cart, so that when the user goes to pay for their items, the server can retrieve the list of items selected by looking at the user's cookie. A cookie only includes text data sent by the web server. This data is never executed as code so it cannot contain programs or viruses, and cannot extract data from your system.
