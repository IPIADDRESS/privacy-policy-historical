Security

Our Site has security measures in place to help protect against the loss, misuse, and alteration of Data under our control. When you access our Site using a browser such as Microsoft Internet Explorer versions 5.5 or higher, Secure Socket Layer (SSL) technology protects information using both server authentication and data encryption to help ensure that Data is safe, secure, and available only to you. Spring Metrics also implements an advanced security method based on dynamic data and encoded session identifications, and hosts the Site in a secure server environment that uses a firewall and other advanced technology to prevent interference or access from outside intruders. Finally, Spring Metrics provides unique user names and passwords that must be entered each time a customer logs in. These safeguards help prevent unauthorized access, maintain Data accuracy, and ensure the appropriate use of Data.
