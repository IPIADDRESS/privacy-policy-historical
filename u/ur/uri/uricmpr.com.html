Privacy Policy

Please read carefully the Privacy Policy below and the Terms of Use before you use Comprsite, use our tools and before conducting any type of transaction with Comprsite. Data Collection

Comprsite does not save or collect any data of User unless User decides to register. However, f User decides to register Comprsite, some personal data such as e-mail address will be required for registration. We do not save any kind of information which is not asked User directly in forms.

Data Protection

Comprsite does not sell or share any kind of data for any kind of purpose to 3rd parties. We do not share your personal information to anyone. Comprsite takes measures for data and code security. We implemented cryptographic functions in order to keep data safety.

Seo And Website Analyses

Comprsite provides online monitoring and analysing web tools, including, but not limited to, the free Seo Tools and Website Tools. The public information such as site title, description, homepage screenshot, most searched queries, domain age, etc. of domains will also be visible to other users. Whenever user types a domain, Comprsite automatically start collecting public information of given domain.
