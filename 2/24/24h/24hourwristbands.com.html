COOKIES

Cookies are files Web browsers place on a computer's hard drive. They are used to help us authenticate clients and provide timesaving shortcuts and preferences. NetBrands uses a persistent encrypted cookie, if the client requests, to save and retrieve individual authentication information as well as other per-client device preferences. When a visitor follows a link to the NetBrands Web site from an affiliate or promotion, NetBrands stores information about that affiliate or promotion in a session cookie. Acceptance of these cookies is optional. NetBrands issues a mandatory session cookie to each user of the Services only to record encrypted authentication information for the duration of a specific session. If you reject this cookie, access to and usage of the Services will be denied.

Customers and visitors must check the policy statement of these others Web sites to understand their policies. Customers and visitors who access a linked site may be disclosing their private information. It is your responsibility to keep such information private and confidential.

SECURITY

NetBrands has security measures in place for the Services to help protect against the loss, misuse and alteration of the data under NetBrands's control. When the Services are accessed using Netscape Navigator, or Microsoft Internet Explorer versions 4.0 or higher, Secure Socket Layer (SSL) technology protects information using both server authentication and data encryption to help ensure data is safe, secure and available only to you. NetBrands also implements an advanced security method based on dynamic data and encoded session identifications and hosts the Services in a secure server environment using a firewall and other advanced technology to prevent interference or access from outside intruders. Finally, NetBrands requires unique user names and passwords that must be entered each time a customer logs on. These safeguards help prevent unauthorized access, maintain data accuracy and ensure the appropriate use of data.
