    3. A. What Are Cookies?
    "Cookies" are a feature of web browser software that allows web servers to recognize the computer used to access a web site. They are small pieces of data that are stored by a user's web browser on the user's hard drive. Information gathered through Cookies and web server logs may include information such as the date and time of visits, the pages viewed, time spent at the Site, and the web sites visited just before and just after our Site. Cookies can remember what information a user accesses on one web page to simplify subsequent interactions with that web site by the same user or to use the information to streamline the user's transactions on related web pages. This makes it easier for a user to move from web page to web page and to complete commercial transactions over the Internet. Cookies should make your online experience easier and more personalized.

    B. How We Use Information We Collect from Cookies.
