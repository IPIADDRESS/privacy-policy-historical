Your continued use of this Website after we make changes is deemed to be acceptance of those changes, so please check the policy periodically for updates.

This online Privacy Policy applies only to information collected through our Website and Services and not to information collected offline.

Our Website allows you to access the following services (“Services”):

  • An online email editor named “BEE Free” that enables users to create responsive email using a drag-&-drop user interface (“BEE Free”);
  • A JavaScript web plugin named “BEE Plugin” that is a web plugin, easily embeddable in other software applications, and used for the purpose to allow users of those applications to create email messages using a drag-&-drop user interface (“BEE Plugin”);
  • An online email editor named “BEE Pro”, that enables users to create, save and edit an email message using a drag-&-drop user interface (“BEE Pro”);
  • An online community that users can use to ask and find information about our Services (“Community”).
  • A newsletter to get regular updates from us (“Newsletter”).

Similarly, we may attempt to detect your geolocation and direct you to a version of the Website that is written in a language that might better match your needs. We reserve the right to maintain, use, and share Anonymous Information without limitation.

We use Information that we collect about you or that you provide to us, including any personal information:
