Information Gathered

KUSports.com gathers identifying information about Website users (“Information”) only when a user provides that information on a voluntary basis such as by registering for or using an Account, Alerts, RSS feeds, by posting User Generated Content, by using social networking tools and the like. These files give a general picture of who is visiting the Website and which pages are being viewed most often. KUSports.com also uses “cookie” technology that allows its servers to deposit codes on a user’s computer. This information helps determine the number of visitors to the Website on an ongoing basis and the types of Internet browsers and operating systems used by the Website’s users. This information is used to enhance your Website experience.
