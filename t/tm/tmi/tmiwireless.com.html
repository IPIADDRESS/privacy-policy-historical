Website Traffic Data

Our website is hosted on a web server that maintains log files that record website traffic data. When a visitor accesses any resource on our website such as a web page, an image, or any other piece of information published on our site, the web server will record various technical information about the visitor's request in this log file. Some information recorded includes the visitor's: time of request, requested URL, IP address, browser type, referring website URL, operating system, and cookie data. We may use the data in these log files to analyze our users' behavior for the purpose of improving the website and the experience in future versions of our website. In the rare cases of suspected fraudulent or criminal activity, we may also use the log file data as needed to help authorities or impacted organizations.
