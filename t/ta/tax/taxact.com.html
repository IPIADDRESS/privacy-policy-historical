Use of Log Files, LiveStats and Cookies

Like most Web servers, 2nd Story Software relies on log files. Log files include: Internet Protocol (IP) addresses, browser type, Internet Service Provider (ISP), number of page views, platform type, date and time stamps, as well as the number visitors to our web site. This information is used by 2nd Story Software to administer the site, analyze trends, track aggregated user movements, and gather broad demographic information for aggregated use only. Our Web site also uses tracking software, which is supplied by LiveStats by DeepMetrix. This software further enables us to anonymously analyze trends occurring on our Web site. Personally identifiable information is not stored by the LiveStats server, as the software does not collect personal information such as name, age, phone number, mailing address or credit card numbers from visitors. For more information about the LiveStats Log Analyzer Software and DeepMetrix's commitment to privacy for the companies who use LiveStats, visit http://www.deepmetrix.com/legal/privacy/products/. For each visitor to our Web page, LiveStats automatically recognizes only the consumer's domain name, IP address, access times and pages viewed. IP addresses, etc. are not linked to personally identifiable information.

TaxACT.com employs the use of cookies. A cookie is a piece of data stored on the user's computer tied to information about the user. Usage of a cookie is in no way linked to any personally identifiable information while on our site. We use both session ID cookies and persistent cookies. For the session ID cookie, once users close the browser, the cookie simply terminates. A persistent cookie is a small text file stored on the user's hard drive for an extended period of time. If users reject the cookie, they will not be able to purchase products from the TaxAct.com website.
