Security Procedures
TRUE recognizes how important and sensitive data is, and that is why we have implemented policies and procedures to preserve the security of your information.  While TRUE strives to protect the information you provide through TRUE.com, no data transmission over the Internet can be guaranteed to be 100% secure.  As a result, while we strive to protect your information, TRUE cannot ensure or warrant the security of any information you transmit to or receive from us through our website and online services.  TRUE.com operates secure data networks that are password-protected and are not available to the public.  When transmitting information between you and TRUE.com, data security is handled through a security protocol called Secured Sockets Layer (SSL).  SSL is an Internet security standard using data encryption and Web server authentication. 

Data encryption is the process of transforming legible information into a complex arrangement of illegible information.  The illegible data is made legible only by a process called "decryption." A mathematical formula known as a "key" is used to encrypt data when it is sent over the Internet, and then to decrypt the data when it is received.  SSL uses a private/public key.  Encryption strength is measured by the length of the key used to encrypt the data; i.e., the longer the key, the more effective the encryption.  Using the SSL protocol, data transmission between you and the TRUE.com server is performed at a standard level of encryption strength. 

An SSL certificate is issued to ensure authentic and accurate identification of a secured Web server.  This translates into a digital ID, or electronic fingerprint, that is included in the key used to encrypt the data.  This ensures that only the intended recipient, TRUE, is able to decrypt the data that you transmit.

TRUE reviews its security policies regarding TRUE.com on a regular basis, and changes are made as necessary.  Only a select group of authorized individuals within TRUE have access to user information.
