What is a cookie?
Cookies are small files which are downloaded on to a device when the user accesses to certain websites. Cookies allow a website to recognize users, to understand the usersâ navigation on different websites.

In practice, the website you visit sends information to the browser which then creates a text file. Every time the user goes back to the same website, the browser retrieves and sends this file to the website's server.
