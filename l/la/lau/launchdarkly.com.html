LaunchDarkly's Feature Flags
Feature flags/toggles/controls are a software development best practice of gating functionality, used by companies like Google and Facebook.

Functionality can be deployed “off”, then turned on via the feature flag, separate from deployment.

LaunchDarkly allows you to set up flags in minutes to manage the entire lifecycle of a feature.
