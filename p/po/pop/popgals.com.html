When visiting any Internet site, your unique address called an "IP address" is recorded. 5mgames.com does not release any information about the collection of this address to any third party except under court order. 5mgames.com archives the log files in order to create aggregate statistical reports, detect errors on the web site, and for security reasons. This is done to verify billing for the ads. No additional information associated with any specific user is provided to the advertisers.
