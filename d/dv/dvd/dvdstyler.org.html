What we collect

On our website, we use the open source web analysis software Piwik. Data collected by Piwik includes IP, time and duration of the visit, what pages are visited, used browser and plugins, search-engines and referrer. While Piwik uses a cookie to distinguish between individual visitors, the collected data doesn't allow to identify individuals. The collected data will only be used to improve the website - especially that data won't be matched with data from other sources, and we won't share the collected data with third parties. We'll only use anoymized, collective analysis provided by Piwik for our evaluation.
