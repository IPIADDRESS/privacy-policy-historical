Security

This site has security measures in place to protect the loss, misuse and alteration of the information under our control.

These measures include, but are not necessarily limited to;

  • Regular (nightly) offsite backups to 2 seperate locations
  • Access to the core server is only allowed using SSH2 technologies
  • Access to SSH2 is only granted to users with previously uploaded public certificates. Password access to the server is forbidden
  • Limited access, even via SSH2, only to people known personally by Atomic Systems IP Ltd staff.
  • Regular monitoring of server logs for suspicious activity
  • Individual SysAdmins are radio paged if anything looks wrong on the system.
  • A plethora of other security measures that we would not wish to divulge!
